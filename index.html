    Blog Entry 1: Quick comparison of python, Anaconda3 and Intel distribution for python:

    I was not able to find a numerical comparison between these three platforms, so I decided to come and try it for myself. I ran these experiments on a Intel(R) Core(TM) i7-3770K CPU @ 3.50GHz model 58 coming from CPU family 6 with stepping = 9, microcode 0x12 at cpu MHz = 1667.327 with a cache size = 8192 KB. and ram total of 32. This is not an exhaustive comparison since I decided to go with a simple calculation of pi. The code that I ran was:

    The code I used

    A simple calculation of pi, I took this code almost exactly from David Bolton's blog.

    from decimal import *
import time

getcontext().prec = 50
s = Decimal(1);   #Sign
pi = Decimal(3);
n = 500000

start_time = time.time()
for i in range (2, n * 2, 2):
    pi = pi + s * (Decimal(4) / (Decimal(i) * (Decimal(i) + Decimal(1)) * (Decimal(i) + Decimal(2))))
    s = -1 * s
print ("\nApproximate value of PI :",pi," for ",n)
print("\nIt took--- %s seconds ---" % (time.time() - start_time))
